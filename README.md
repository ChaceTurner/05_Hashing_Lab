05_Hashing_Lab
==============

Implement a hash table in C++

Requirements
------------

1. `keyExists`, `find`, `remove`, and `size` should all be O(1)
2. `add` should be O(1), except for the call to `grow`.
3. `grow` should be O(n)
4. Do not leak memory
